# whatinthePI - Raspberry Pi Tools Collection

A comprehensive set of tools for Raspberry Pi Wi-Fi management and Access Point setup.

## 📚 Available Tools

| Tool | Description | Quick Command |
|------|-------------|---------------|
| **AP Setup** | Create your own Wi-Fi access point | `apsetup` |
| **Wi-Fi Manager** | Interactive menu for Wi-Fi control | `wifiman` |
| **Wi-Fi Helper** | Quick command-line Wi-Fi control | `wifi [command]` |
| **System Status** | View Pi system information | `status` |
| **Help System** | Show all available commands | `help` |
